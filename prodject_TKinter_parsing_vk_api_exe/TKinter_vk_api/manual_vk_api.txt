ЗНАКОМСТВО С ПРОГРАММОЙ:
Для корректной работы программы и для облегчения работы пользователя, все ключевые файлы и папки
находятся в скрытом режиме(чтобы их открыть смотрите текущие настройки вашей системы) оставленны только,
файл для настройки - data_api_vk.py и файл запуска - TKinter_vk_api.exe.

1. Файлы находящиеся в программе:
	А) TKinter_vk_api - непосредственно папка с нахождением приложения
		а) data_api_vk.py - файл настройки парсера
		б) TKinter_vk_api.exe - файл для запуска приложения  

2. Запуск программы:
	А) Проходите в директорию ...project_TKinter_vk_api\TKinter_vk_api и запускаете TKinter_vk_api.exe

3. Работа с программой:
	A) После появления диалогового окна выбираете нужный вам пункт.
		а) Кнопка - "НАЧАТЬ ПАРСИНГ", начинает парсинг с настройками по умолчанию,
		где настройки по умолчанию - это ваши последние настройки в файле data_api_vk.py.
		б) Кнопка - "ОТКРЫТЬ MANUAL" открывает данное руководство.
	Б) После нажатия кнопки "НАЧАТЬ ПАРСИНГ", начнется парсинг и файл с данными в формате "CSV"
	(он же EXEL файл) и именем ID альбома, появиться в директории(папке нахождения) программы.
	примечание: не закрывайте диалоговое окно и команндную строку виндовс до конца парсинга(иначе это прервет процесс)

4. Настройка программы:
	А) Что можно настроить в программе.
		а) ID альбома группы.

	Б) Как поменять альбом для парсинга:
		а) Проходите в директорию ...project_TKinter_vk_api\TKinter_vk_api и с помощью блокнота,
		откройте файл data_api_vk.py
		б) Необходимо выбрать строчку "ALBUM_ID = '279802653'  # настраивать альбом тут, помещать данные ID альбома между кавычек"
		в) В данной строчке необходимо поменять значение ID НАПРИМЕР:
								       БЫЛО: ALBUM_ID = '279802653' - где 279802653 евляется ID альбома
                                                                      СТАЛО: ALBUM_ID = '101010101' - где 101010101 евляется ID альбома
                г) Далее просто закрываете файл, в диалоговом окне выбираете сохранить изменения.
		д) Далее переходите к пункту 2 и 3.Б данного руководства и получаете удовольствие от сохраненного времени))

5. Дополнительная информация:
        А) Когда вы меняете настройки ID альбома, после парсинга формируется дополнительный файл "CSV" с названием ID альбома.
		а) Если в заявленном альбоме для парсинга, отсутствуют коментарии, то файл с данными не будет формироваться!!
	Б) Данные каждого альбома формируются в отдельный "CSV" файл
	В) Парсер рассчитан только на одно сообщество в VK а именно https://vk.com/optovye, к другим сообществам он не применим 
	Г) Чтобы отановить парсинг(закрыть программу или прервать её) просто кликните по крестику закрытия))
